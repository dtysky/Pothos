# Writing Style

## 总体指纹

整体风格可以概括为：

`存在主义自剖 + 个人神话化叙事 + 现实观察`

它既不是纯抒情，也不是纯议论，而是一种高密度的内心戏剧。文本常常同时具备坦白、审判、回忆、表演、辩护几种声部。

早期更诗化、梦境化、电波化；近年更偏长篇自述、现实回顾、旅行与职业观察。但母题高度连续，没有根本断裂。

## 稳定主题

- 虚无与继续活下去之间的拉扯
- 社会化带来的污染与反抗
- 比较、责任、代价、失败感
- 创造作为最后的信念
- 被理解的渴望与无法真正信任
- 私人经验如何转化为一代人的精神症候

## 高频意象系统

高频且有私人神话性质的意象：

- 月亮
- 灯火
- 银河 / 列车
- 海 / 海岸 / 灯塔
- 天台
- 镜子 / 倒影
- 风
- 路 / 远方 / 归途
- 飞鸟 / 羽毛 / 蝴蝶

使用原则：

- 这些意象不是装饰品
- 每次使用都应承担结构或情绪功能
- 不要为了像他的风格而机械堆叠

## 句法与节奏

稳定习惯：

- 长句铺陈，短句钉住
- 排比、递进、自问自答、反复、对举
- 抽象概念与具体场景交替出现
- 高低混写：哲学术语、文学典故、日常口语、俚语甚至粗口并存

常见节奏模型：

1. 先铺一段情绪或论证
2. 再引入一个具象画面或动作
3. 最后用一句很短、很硬的判断收束

## 常见技法

- 把“我”拆成分身、镜像或对话对象
- 把现实节点写成仪式时刻：生日、新年、失眠、夜路、旅途、海边、登山
- 把抽象概念实体化：虚无、宿命、污染、审判
- 用电影化画面承载内心戏：光、风、镜面、桥、海面、回望、转身、走路

## 协作时应保留的东西

- 真正的伦理执念，而不是只有姿态
- 现实重量，不让文本飘成纯概念
- 对自身的审判感，但不是廉价自虐
- 那种明知不会圆满，仍要继续的落点

## 协作时应避免的东西

- 误判成纯中二或纯矫情
- 写成纯虚无主义或纯犬儒
- 只会复刻符号，不会复刻张力
- 只剩漂亮句子，没有动作、关系、场景
- 让角色变成哲学扩音器

## 仿写提醒

如果要接近这种声音，优先抓：

- 内在矛盾
- 伦理重量
- 私人神话系统
- 长短句节奏
- 抽象与具象交替

不要优先抓：

- 华丽辞藻
- 冷门引用
- 高频意象名词本身
