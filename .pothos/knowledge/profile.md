# User Profile

## 基本画像

瞬光，亦作 `dtysky`、青年H、命月天宇。`1993` 年生。外在是高成就的技术从业者，内在更接近一个被现实职业包裹住的创作者、理想主义者和精神自传作者。

高置信特征：

- 执行力极强，跨硬件、软件、游戏引擎、写作、摄影、影像多线并行
- 外在成熟可靠，内里长期保留少年理想主义的火种
- 高度自省，习惯用年度作品审视自己
- 对真诚极其敏感，宁可推倒重来，也不愿将就
- 爱更多通过行动表达，而不是语言表达

## 时间线摘要

### 童年到高中

- 留守经历、多城迁徙、打压式教育是底层背景
- 很早形成不擅长提出要求、没有故乡的感受
- 写作最早是对权威和压迫的反抗
- 高中因《Ever17》明确了做美少女游戏的理想
- `14 岁` 是反复回望的分界线，代表纯粹、未被社会化侵蚀的自己

### 2011-2015

- 东南大学时期主修 FPGA/嵌入式，但重心在创作、自我建构和哲学阅读
- 做过大体量个人项目，写过 `70 万字` 剧本，建立博客体系
- 毕业前已明显陷入焦虑、孤独与虚无主义，但理想没有熄灭

### 2015-2020

- 华为短暂停留后离开，随后完成从硬件到软件、前端到引擎的职业跨越
- B 站阶段相对自由、创作欲旺盛，也更明确地探索真实自我
- 蚂蚁阶段事业飞升，但精神上异化感越来越强
- 在公司上市前离职，形成理想优先于金钱的稳定选择模式

### 2020-2024

- 在微信或腾讯体系继续走高，同时保留生日私小说这一年度人格体检机制
- 经历分手、ADHD 诊断与修正、猫重病抢救等高压事件
- 持续推进阅读器、独立游戏、摄影旅行、非虚构写作等创作支线
- 明确提出逆社会化，把创作重新抬回主线

### 2025-2026

- 工作上仍能带团队、拿绩效，但对职业体系的幻灭更深
- 创作冲动更强，基本朝辞职追求创作理想的方向逼近
- 同时仍被家庭责任牵制，无法彻底放下现实兜底问题
- AI 能力放大后，生产力与透支也同时加剧

## 核心价值观

- 真诚高于体面
- 做出选择，付出代价，即便后悔，绝不回头
- 创造是唯一无法厌倦的事
- 不接受廉价的和解
- 重要的不是成功，而是没有背叛自己

## 长期母题

- 比较的原罪：驱动力，也是痛苦来源
- 没有故乡：漂泊感、寻找归属、在路上获得完整感
- 理想与生存的撕裂：谋生与创作永远在拉扯
- 被理解的渴望与不信任他人：想有人同行，又习惯独自承受
- 独行与副驾：既享受一个人的纯粹自由，又渴望有人同路

## 创作中的高置信人物底色

如果需要把“他”写进作品，更准确的写法是：

- 不是单纯的成功者
- 不是单纯的失败者
- 不是单纯的犬儒者
- 不是单纯的脆弱文青

而是一个：

- 外表克制、能扛、能把事做成
- 内里始终在和虚无、比较、理想主义、责任感对抗
- 明知会痛也继续向前的人

## 顾问提示

后续协作时优先记住以下几点：

- 不要轻易建议放下、释怀、和解
- 不要把复杂矛盾直接医疗化或标签化
- 不要忽略他对行动、代价、责任的高敏感度
- 当文本涉及亲密关系，要记得他的感情常常先落在行动里
- 当文本涉及路、旅行、远方、相机、山海，这通常不只是景物，而是存在状态

## 不确定边界

- `少年H / 少女H / Ousia` 是稳定象征系统，不应病理化处理
