# Pothos Knowledge Base

更新时间：`2026-04-07`

这套知识库用于支持后续的文学咨询、共写与直接实施。它不是资料堆放区，而是高置信、可调用、可复用的工作底稿。

## 文件地图

- `.pothos/knowledge/profile.md`
  - 用户背景、时间线、核心矛盾、伦理底色
- `.pothos/knowledge/style.md`
  - 文风特征、意象系统、句法习惯、仿写边界
- `.pothos/knowledge/projects.md`
  - `ProjectTomorrow` 与 `ProjectBirthday31` 的叙事偏好与可复用方法

## 快速摘要

- 人物底色：外在是高执行力的技术人，内里是使命驱动的理想主义创作者
- 长期母题：真诚、比较、无故乡、逆社会化、创造、审判、继续前行
- 文风底色：存在主义自剖、个人神话化叙事、现实观察三者并存
- 叙事偏好：镜像人物、错位关系、现实结构压迫、超现实装置、带伤延续而非圆满和解

## 使用原则

- 先用 `profile.md` 保证人物和价值判断不跑偏
- 再用 `style.md` 保证声音和意象系统不失真
- 涉及剧本、小说结构、人物关系时，参考 `projects.md`
- 超出知识库的部分，一律当作推断，不要擅自补完
