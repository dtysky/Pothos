# 纪录片结构

## 总体原则

- 以地理推进作为骨架：从现在的生活地点出发，沿着现实路线一步步逼近出生地
- 以问题推进作为刀口：每一段都围绕一个关于"根"的问题展开，而不是只记录旅程
- 以现在时为主体：不是回忆录，而是一次正在发生的验证
- 允许答案失败：结构不能预设和解，也不能把抵达当作天然高潮

## 建议结构

### 第一段：带着一句话上路

主问题：为什么一个一直没有故乡感的人，会突然决定返乡？

这一段先交代当下的生活状态、漂泊经验，以及"落叶归根"这四个字为何在此刻重新变得刺耳。它不是一个天然成立的愿望，而更像一句从文化里继承来的命令，一个悬在头顶的问题。

这一段的作用是立住出发点：他不是去找温暖，而是去找答案。

### 第二段：逆着自己的迁徙史走

主问题：人的根是在出生地，还是在一路塑形他的那些地方？

旅程中经过的城市、车站、道路、转运节点，可以成为自我史的切片。每到一个地方，都不是单纯报地名，而是要问：这里在我身上留下了什么？如果一个人是被许多地方共同制造出来的，那"根"还能不能只指向单一坐标？

这一段让片子从单点返乡，变成一张迁徙地图上的逆流运动。

### 第三段：越接近，越不确定

主问题：当出生地越来越近时，期待究竟在增长，还是在崩塌？

这一段拍"接近"本身。地理距离缩短，心理距离却未必缩短。可以逐渐引入更具体的信息：家人说法、户籍记录、旧地址、地图坐标、地方名称。但所有信息都不直接通向归属，反而让问题变得更尖锐。

这里需要形成一种悬置感：抵达不再是答案，更像一次审判前的沉默。

### 第四段：抵达，然后彼此失认

主问题：那个地方认不认我？我认不认它？

这是全片的核心段落。抵达出生地之后，最重要的不是拍"我终于到了"，而是拍辨认失败的瞬间。可能是地貌变了，可能是房子没了，可能是地方还在但关系断了，可能是你站在那里却没有任何身体性的确认感。

这一段要让观众真正看到：返乡并不是人与故乡天然相认，而可能是一场双向失认。地方没有义务认你，你也未必还能诚实地认它。

### 第五段：离开时，重新理解"根"

主问题：如果根不在那里，或者从来不在那里，人还能怎样理解自己的来处？

结尾不追求温暖收束，也不必给出哲学结论。更合适的落点是：承认故乡可能不是一个可返回的实体，承认出生地未必等于归属，但仍然接受这趟旅程的必要性。

返乡的意义，不是成功回到树下，而是亲自确认那棵树是否存在过，以及自己如今究竟由什么构成。

## 叙事手段建议

- 旁白可以存在，但应以提问、修正、反驳自己为主，避免把片子写成结论先行的哲学讲演
- 采访可以作为镜面，而不是证词收集；家人、当地人、同样漂泊的人，最好分别对应不同版本的"根"
- 路上的空镜、候车、过路、停顿都很重要，它们不是填充，而是让"接近"和"不确定"有身体感
- 可以允许片中出现信息错误、记忆偏差、期待落空，因为这些恰恰构成主题本身

## 当前版本最该守住的东西

- 不把返乡浪漫化
- 不把失败当成素材消费
- 不过早替观众总结答案
- 让"抵达"成为问题最锋利的时刻，而不是问题结束的时刻
