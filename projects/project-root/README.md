# ProjectRoot

一个在中国土生土长，却一直漂泊、没有故乡的人，从他现在的地方出发，穿越大半个中国，回到他出生的地方。

## 基本信息

- **代号**：ProjectRoot
- **输出形态**：纪录片
- **阶段**：概念设计
- **核心命题**：落叶归根——但那棵树还在不在？你是不是早就不是从那棵树上掉下来的叶子了？

## 核心张力

这不是一个文化意义上的寻根。不是海外华人回望故土，不是知识分子追溯传统。这是一个在自己的国土上从未扎下根的人，沿着自己的迁徙路线逆流而上，回到那个物理意义上的起点。

他带着"落叶归根"这四个字上路，但这趟旅程真正要回答的是：那片土地认不认他？他认不认它？根是在那里等着他，还是从来就不存在？

## 目录说明

- `design/` — 项目设计：概念、路线、结构、参考
- `notes/` — 咨询归档
- `research/` — 地理、历史、家族调研
- `scripts/` — 纪录片脚本
- `shots/` — 镜头设计、视觉概念
- `journal/` — 旅途日志
- `fragments/` — 独立文字片段、旁白草稿
